# ⚡ AutoKeyButton

AutoKB 是一款功能强大的桌面自动化脚本工具，旨在通过智能输入录制、回放和宏管理来简化重复性任务。基于 Rust 和 Tauri 构建，它能够提供现代化的毛玻璃界面以及极低延迟的操作体验。

---

## ✨ 核心功能

### 🔴 智能录制
- **全局捕获**：支持录制全系统的键盘和鼠标事件，而不仅仅局限于应用窗口内。
- **智能过滤**：自动处理按键重复和复杂的输入序列，确保脚本回放的高度还原。
- **状态感知**：通过全局屏幕边框悬浮窗实时显示录制状态，确保用户随时掌握录制进度。

### ⏯️ 高精度回放
- **毫秒级还原**：以毫秒级的精度重现已录制的动作，确保自动化流程的稳定性。
- **播放速度调节**：支持实时调整执行倍速，无论是慢速调试还是极速运行都能轻松应对。
- **控制提示**：回放期间开启“蓝色模式”屏幕边框提示，防止手动操作与自动化流程发生冲突。

### 🔗 进阶宏管理
- **输入映射**：将复杂的动作序列映射到单个触发键或鼠标按键上。
- **全局监听**：一键开启或关闭全局宏监听器，灵活切换工作模式。
- **持久化存储**：脚本与宏设置自动保存至本地，随时随地快速调用。

### 📝 脚本细微调优
- **事件列表**：通过列表直观查看捕获的所有事件及其时间轴。
- **即时编辑**：直接在界面中修改事件间的延迟时间或删除不必要的步骤。
- **批量缩放**：一键按比例缩放脚本内所有延迟，快速调整整体执行节奏。

---

## ⌨️ 快捷控制

无需切换窗口，即可快速管理自动化流程：

| 功能 | 快捷键 |
| :--- | :--- |
| **开始/停止 录录制** | `F9` |
| **开始/停止 回放** | `F10` |

---

## 🎨 专业界面设计
- **深色模式优先**：采用沉浸式深色主题，专注高效。
- **实时状态反馈**：动态指示灯清晰展示当前引擎状态（录制中、回放中或宏监听激活）。
- **系统托盘集成**：支持最小化至托盘后台运行，点击即可快速唤回。

---

## 🚀 快速上手

1. **启动 AutoKB**。
2. 按下 **`F9`** 开始录制您的操作流程。
3. 执行您希望自动化的任务。
4. 再次按下 **`F9`** 结束并保存脚本。
5. 在**事件列表**中对操作进行微调，或直接按 **`F10`** 测试回放效果。
