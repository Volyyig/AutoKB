# ⚡ AutoKeyButton

AutoKB 是一款基于 **Tauri v2** 和 **Vue 3** 开发的轻量级桌面自动化工具。它支持实时录制、编辑和回放键盘/鼠标操作，并通过全局热键管理复杂的宏指令。

## ✨ 功能特性

- **🔴 高精度录制**: 实时捕捉键盘和鼠标事件。
- **▶️ 灵活回放**: 支持自定义播放速度和延迟时间。
- **⌨️ 全局热键**: 快捷控制，无需切换窗口即可触发自动化。
- **🔗 宏管理**: 将特定触发键（按键/鼠标）映射为复杂的操作序列。
- **📊 事件编辑器**: 精细化调整步骤、修改延迟或删除多余事件。
- **🖥️ 极简叠加层**: 录制和回放期间提供视觉反馈，不干扰正常操作。
- **🌓 现代 UI**: 专注效率的暗色系界面设计。

## 🛠️ 技术栈

- **后端**: Rust, [Tauri v2](https://tauri.app/)
- **前端**: [Vue 3](https://vuejs.org/), TypeScript, [Vite](https://vitejs.dev/)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **依赖库**: `rdev` (事件采集), `enigo` (事件模拟)

## ⌨️ 全局热键

| 按键 | 功能 |
| :--- | :--- |
| **F9** | 开始 / 停止录制 |
| **F10** | 播放 / 停止脚本 |
| **Esc** | 紧急停止 (强制停止所有录制和回放) |

## 🚀 快速上手

### 环境准备

- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js](https://nodejs.org/) (建议使用 `pnpm` 或 `npm`)

### 开发环境

1. 克隆仓库:
   ```bash
   git clone https://github.com/your-repo/autokb.git
   cd autokb
   ```

2. 安装依赖:
   ```bash
   npm install
   ```

3. 启动开发模式:
   ```bash
   npm run tauri dev
   ```

### 编译打包

生成生产环境安装包:
```bash
npm run tauri build
```

## 📖 使用指南

1. **录制**: 按下 `F9` 或点击录制按钮。主窗口会隐藏并显示红色叠加层。执行操作后再次按下 `F9`。
2. **编辑**: 在“事件列表”中查看捕获的事件，可以调整延迟或删除步骤。
3. **回放**: 按下 `F10` 或点击播放。蓝色叠加层表示正在回放。
4. **宏**: 使用宏面板创建自定义映射，例如通过单击一个键触发一系列复杂的组合操作。
